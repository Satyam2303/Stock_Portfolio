# stockscope

A beginner-friendly Python package to:
- Download stock price data (yfinance)
- Compute indicators (returns, SMA, EMA, RSI)
- Compute risk metrics (volatility, max drawdown, historical VaR)
- Analyze simple portfolios (weights, buy-and-hold values)
- Visualize results (matplotlib)
